# CDS Mobile Playground

A React Native/Expo mobile application playground for prototyping with Coinbase Design System (CDS) components.

## 🚀 Getting Started

From the root directory:

```bash
yarn install
yarn mobile:dev
```

Or from this directory:

```bash
yarn dev
```

This will start the Expo development server. You can then:

- Press `i` to open iOS simulator
- Press `a` to open Android emulator
- Scan the QR code with Expo Go app on your device

## 📱 Demo App

The current app is a simple Expo blank template. This serves as a starting point for building mobile prototypes with CDS components.

## 🔧 Building Your Own Mobile Prototype

This is a standard Expo app, so you can:

1. **Edit `App.js`** to create your mobile prototype
2. **Add CDS mobile components** (when available)
3. **Use standard React Native components** for now

## 📝 Available Scripts

- `yarn dev` - Start Expo development server
- `yarn start` - Start Expo development server
- `yarn android` - Start on Android emulator
- `yarn ios` - Start on iOS simulator
- `yarn web` - Start mobile app in web browser

## 📋 Prerequisites

- **iOS Development**: Xcode and iOS Simulator
- **Android Development**: Android Studio and Android Emulator
- **Physical Device**: Expo Go app from App Store/Play Store
